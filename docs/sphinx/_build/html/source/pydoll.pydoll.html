pydoll.pydoll.element module

class pydoll.pydoll.element.WebElement(object_id: str, connection_handler: ConnectionHandler, method: str = None, selector: str = None, attributes_list: list = [])[source]

Bases: FindElementsMixin

Represents a DOM element in the browser.

This class provides methods to interact with and retrieve information about DOM elements. It allows operations such as clicking, sending keys, getting attributes, and other common web element interactions. It inherits element finding capabilities from FindElementsMixin.

property bounds: list
