class pydoll.pydoll.mixins.find_elements.FindElementsMixin[source]

Bases: object

A mixin class that provides element finding and waiting capabilities.

This mixin provides methods for finding elements in the DOM using various selector strategies, waiting for elements to appear, and interacting with elements. Classes that include this mixin will gain the ability to locate elements in web pages.

async find_element(by: Literal[By.CSS_SELECTOR, By.XPATH, By.CLASS_NAME, By.ID, By.TAG_NAME], value: str, raise_exc: bool = True)[source]

Finds an element on the current page using the specified selector.

This method locates the first element matching the given selector and returns a WebElement instance representing that element. If no element is found, it either raises an exception or returns None, depending on the raise_exc parameter.

Parameters:

  • by (SelectorType) – The type of selector to use (e.g., ‘css’, ‘xpath’).

  • value (str) – The value of the selector to locate the element.

  • raise_exc (bool) – Whether to raise an exception if the element is not found. Defaults to True.

Returns:

The found element as a WebElement instance, or

None if no element is found and raise_exc is False.

Return type:

WebElement or None

Raises:

ElementNotFound – If the element is not found and raise_exc is True.

async find_elements(by: Literal[By.CSS_SELECTOR, By.XPATH, By.CLASS_NAME, By.ID, By.TAG_NAME], value: str, raise_exc: bool = True)[source]

Finds all elements on the current page using the specified selector.

This method locates all elements matching the given selector and returns a list of WebElement instances. If no elements are found, it either raises an exception or returns an empty list, depending on the raise_exc parameter.

Parameters:

  • by (SelectorType) – The type of selector to use (e.g., ‘css’, ‘xpath’).

  • value (str) – The value of the selector to locate the elements.

  • raise_exc (bool) – Whether to raise an exception if no elements are found. Defaults to True.

Returns:

A list of WebElement instances representing the

found elements. Returns an empty list if no elements are found and raise_exc is False.

Return type:

list[WebElement]

Raises:

ElementNotFound – If no elements are found and raise_exc is True.

async wait_element(by: Literal[By.CSS_SELECTOR, By.XPATH, By.CLASS_NAME, By.ID, By.TAG_NAME], value: str, timeout: int = 10, raise_exc: bool = True)[source]

Waits for an element to be present in the DOM.

This method repeatedly attempts to find an element until it is found or the timeout is reached. It is useful for handling dynamic content that may not be immediately available.

Parameters:

  • by (SelectorType) – The type of selector to use (e.g., ‘css’, ‘xpath’).

  • value (str) – The value of the selector to locate the element.

  • timeout (int) – Maximum time in seconds to wait for the element. Defaults to 10 seconds.

  • raise_exc (bool) – Whether to raise an exception if the element is not found within the timeout. Defaults to True.

Returns:

The element found in the DOM, or None if

not found and raise_exc is False.

Return type:

WebElement or None

Raises:

TimeoutError – If the element is not found within the timeout and raise_exc is True.
